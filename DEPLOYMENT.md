# Deployment Guide - Leveler Application

## Server Information
- **Server IP**: 75.119.139.18
- **Target Directory**: /var/www/leveler
- **Repository**: https://github.com/amithyone/leveler

## Step-by-Step Deployment Instructions

### Option 1: Using the Deployment Script

1. **Connect to your server via SSH:**
   ```bash
   ssh root@75.119.139.18
   # or
   ssh your_username@75.119.139.18
   ```

2. **Upload the deployment script:**
   - Copy `deploy.sh` to your server, or
   - Create it directly on the server

3. **Make it executable and run:**
   ```bash
   chmod +x deploy.sh
   sudo ./deploy.sh
   ```

### Option 2: Manual Deployment

**Step 1: Connect to Server**
```bash
ssh root@75.119.139.18
```

**Step 2: Navigate to /var/www**
```bash
cd /var/www
```

**Step 3: Clone Repository (if first time)**
```bash
git clone https://github.com/amithyone/leveler.git
cd leveler
```

**Or Update Existing (if already cloned)**
```bash
cd /var/www/leveler
git pull origin main
```

**Step 4: Install Dependencies**
```bash
composer install --no-dev --optimize-autoloader
```

**Step 5: Set Permissions**
```bash
sudo chown -R www-data:www-data /var/www/leveler
sudo chmod -R 755 /var/www/leveler
sudo chmod -R 775 /var/www/leveler/storage
sudo chmod -R 775 /var/www/leveler/bootstrap/cache
```

**Step 6: Create Storage Link**
```bash
php artisan storage:link
```

**Step 7: Configure Environment**
```bash
cp .env.example .env
nano .env  # Edit with your production settings
```

**Step 8: Generate Application Key**
```bash
php artisan key:generate
```

**Step 9: Run Migrations**
```bash
php artisan migrate --force
```

**Step 10: Seed Database (Optional)**
```bash
php artisan db:seed
```

**Step 11: Clear and Cache**
```bash
php artisan config:clear
php artisan cache:clear
php artisan route:clear
php artisan view:clear
php artisan config:cache
php artisan route:cache
php artisan view:cache
```

## Web Server Configuration

### For Apache

Create virtual host configuration:

```apache
<VirtualHost *:80>
    ServerName yourdomain.com
    ServerAlias www.yourdomain.com
    
    DocumentRoot /var/www/leveler/public
    
    <Directory /var/www/leveler/public>
        AllowOverride All
        Require all granted
    </Directory>
    
    ErrorLog ${APACHE_LOG_DIR}/leveler_error.log
    CustomLog ${APACHE_LOG_DIR}/leveler_access.log combined
</VirtualHost>
```

Enable the site:
```bash
sudo a2ensite leveler.conf
sudo systemctl reload apache2
```

### For Nginx

Create configuration:

```nginx
server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;
    root /var/www/leveler/public;

    add_header X-Frame-Options "SAMEORIGIN";
    add_header X-Content-Type-Options "nosniff";

    index index.php;

    charset utf-8;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location = /favicon.ico { access_log off; log_not_found off; }
    location = /robots.txt  { access_log off; log_not_found off; }

    error_page 404 /index.php;

    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.2-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }

    location ~ /\.(?!well-known).* {
        deny all;
    }
}
```

Enable the site:
```bash
sudo ln -s /etc/nginx/sites-available/leveler /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## Troubleshooting

**Permission Issues:**
```bash
sudo chown -R www-data:www-data /var/www/leveler
sudo chmod -R 755 /var/www/leveler
```

**Storage Issues:**
```bash
php artisan storage:link
sudo chmod -R 775 storage bootstrap/cache
```

**Cache Issues:**
```bash
php artisan optimize:clear
php artisan optimize
```
